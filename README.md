<h3 align="center"> 👋 Hi there 👋 </h3>
<p align="center">
안녕하세요 🌱 <br>
Java/Spring 기반 백엔드 개발자로서 게임 퍼블리싱 플랫폼, 캠페인·광고 시스템, 제조 MES까지 다양한 도메인의 서비스를 구축·운영해 왔습니다. 

Spring Boot와 Next.js 기반의 게임 퍼블리싱 플랫폼을 신규로 설계하며 Serverless Web + MSA 아키텍처를 적용했고, OAuth 2.0 인증, TOTP 기반 2차 인증, Redis 세션 클러스터링 등 보안과 확장성을 고려한 인증 체계를 구축했습니다. 또한 CI/CD, 모니터링, 설정 중앙화, HA 구성 등 운영 안정성을 강화하고, 대규모 데이터 테스트와 쿼리 튜닝을 통해 API 응답 속도를 30초에서 1초 미만으로 개선한 경험이 있습니다.

개발 표준과 협업 문화 정립에도 주도적으로 참여하여 Git 브랜치 전략, 코드 리뷰 기준, 아키텍처 가이드 등을 수립했고, 데이터 분석 관점에서는 GA4, Tableau 기반의 지표 분석 환경을 구축해 서비스와 비즈니스 의사결정을 지원했습니다. 

다양한 프로젝트를 경험하며, 변화하는 요구사항 속에서도 안정성과 확장성을 갖춘 시스템을 설계하고 문제를 구조적으로 해결하는 역량을 꾸준히 발전시켜 왔습니다.
</p>
